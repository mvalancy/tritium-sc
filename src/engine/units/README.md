# Engine: Units

Unit type registry for simulation and real-world asset types.

## Files

| File | Purpose |
|------|---------|
| `base.py` | UnitType base class and unit type definitions (16 types) |

## Unit Types

Turret, drone, rover, camera, sensor, relay, human, vehicle, robot,
hostile, neutral, civilian, animal, mesh_radio, and custom types.
Each type defines behavior patterns, capabilities, and rendering hints.
